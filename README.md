# dipper_analysis
data analysis code for project dipper

The bash script run_3dREMLfit.sh, run_grouplevel.sh, and generate_clusters.sh conducts univariate activation analysis using AFNI.

The complete_rsa.ipynb script extracts trial-level activation pattern for a predifined ROI, conducts content-specific Representation Similarity Analysis, and conducts statsitcal tests.
